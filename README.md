# Про сам проект
Проект представляет собой B2B площадку для оптовой продажи различных товаров.
Юр лица могут выкладывать свои товары на площадку, другие юр лица имеют возможность заказывать эти товары.
При закрытии заказа (то есть успешном его выполнении) создаются отчеты, которые будут всегда доступны на странице заказа в истории заказов.

НА ПОДУМАТЬ: 
Список товаров на продажу находится в таблице product_directory, если товара не находится, есть возмжоность создаьт свой товар, который после одобрения администратором будет добавлен в итоговый список

# Про сервисы 

## Order service 
Это сервис для работы с заказами, создавать, отслеживать статус 

## Items service
Сервис для работы с товарами на складе 

## Company service 
Сервис для регистрации

## Report service 
Сервис для генерации отчетов после завершения заказов 

## Email service 
Сервис для email уведомления покупателя и продавца 

## Fake delivery service
Фейковый сервис доставки
